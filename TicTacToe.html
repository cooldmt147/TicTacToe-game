Tic Tac Toe

Instructions: Tic Tac Toe is a game for two players: X & O. The players take turns placing their symbol (X or O) on a 3x3 board. The objective of each player is to have 3 of his symbols on the board in a row, column or diagonal. My game can be played either player vs. player or player vs. computer.
My Algorithm:This tells how the computer will play in certain situations. If an option is not available, the computer will try the one below it.
  • if this is the first turn & computer is X, plays in a corner (*best opening move)
  • if this is the second turn & computer is O, plays in the center (*best second move)
  • if possible, wins
  • if possible, blocks opponent's win
  • if possible, makes a fork
  • if possible (without losing next turn), blocks an opponent's fork
  • if none of these is possible, makes a random move
Note: The game is still under development. I apologize for any bug/problem in the game. Version 2.7.1
